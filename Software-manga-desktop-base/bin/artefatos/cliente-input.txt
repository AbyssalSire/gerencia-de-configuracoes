Levantamento de requitos - Input do cliente


Através de formulários de pesquisa e conversa com o cliente foi possivel filtrar os requisitos de software necessarios para o desenvolvimento do 
seguinte projeto. Nos levamentos de requsito, necessita-se de um sistema que permita que o usuário adicione um mangá a uma lista de mangás
com o objetivo de se manter atualizado com os lançametos de cada episódio do mesmo, e também, adicione comentários sobre o mesmo, 
tudo baseado nos gostos pessoais de cada usuário da aplicação;
Para que tal sistema funcione adequadamente, é necessários os seguintes requisitos:

 - O software permita a adição de um mangá
 - O software permita a exclusão de um mang
 - O software permita a atualização de um mangá
 - O software permita a listagem de mangás
  - O software permita a visualização de um mangá em detalhes.
